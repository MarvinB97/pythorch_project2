# pythorch_project2

## Instalción de dependencias (en modo admin)

En el caso de pytorch se debe seguin las indicaciones de la pagina oficial:
* https://pytorch.org/get-started/locally/

Para la instalción de numpy:
* pip install numpy

Para las graficas (pip o pip3):
* pip install matplotlib

nota: si es en windows revisar la configuracion de las variables (los path).


## Matriz de confusion es necesario:
* pip install scikit-learn
